Aqua World React

A simple and interactive React application that showcases different water vehicles with a clean UI and dynamic routing.

Features
1.Separate Home Page with welcome message
2.Navigation using React Router
3.Display of multiple water vehicles
4.Click on image to view detailed information
5.Clean and responsive grid layout
6.Go Back navigation support

Built With
React
React Router DOM
CSS3
Vite

Water Vehicles Included

* Cruise Ship
* Sail Boat
* Yacht
* Speed Boat
* Fishing Boat
* Cargo Ship

Each vehicle contains:

Image
Color
Description
